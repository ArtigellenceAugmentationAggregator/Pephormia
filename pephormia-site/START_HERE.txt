PEPHORMIA — THE SITE.  This folder is the whole website. Nothing else.

TO PUT IT ON GITHUB (one time):
1. Create a new GitHub repo (empty).
2. Upload the CONTENTS of this folder to it (or: git init, add, commit, push to main).
3. GitHub repo → Settings → Pages → Source = "GitHub Actions".
4. Hostinger DNS for pephormia.com:
     A  @   185.199.108.153
     A  @   185.199.109.153
     A  @   185.199.110.153
     A  @   185.199.111.153
     CNAME  www   <your-github-username>.github.io
5. Every push to "main" auto-builds and goes live. Done.

TO PREVIEW ON YOUR PC:
   npm install
   npm run dev      -> open http://localhost:4321
   (do NOT double-click files in /dist — CSS won't load that way)

TO ADD A BLOG POST:  drop a .md file in  src/content/blog/  and push.
TO ADD A DOWNLOAD:   drop the PDF in  public/downloads/  + a .md in  src/content/library/

That's it. Full detail in README.md.
