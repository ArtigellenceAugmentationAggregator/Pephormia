THE SIGNATURE SYSTEM

Every claim, graded honestly.

We don't invent claims — we aggregate peer-reviewed research and tag each finding by how strong the evidence really is. This is the four-tier evidence box you'll see across the whole platform.

STRONG

WEAK

MECHANISTIC

Preclinical / in-vitro only — mechanism, not proven human outcome.
